# 🚀 QuiKz - AI-Powered Test Constructor

**QuiKz** — мұғалімдерге тест құрастыруды жеңілдететін және оқушылардың білімін бағалайтын веб-платформа.

## 🛠 Технологиялық стек:
* **Frontend:** React.js + Vite
* **Backend:** Node.js + Express.js
* **Database:** PostgreSQL
* **Libraries:** Axios (API), XLSX (Excel export)

## 📖 Жобаны іске қосу:

1. **Backend:**
   ```bash
   cd server
   npm install
   node server.js
2. **Frontend:**
    cd client
    npm install
    npm run dev
